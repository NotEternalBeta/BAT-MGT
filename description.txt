    Описание принципа работы программы по смене режимов электропитания ноутбука под управлением ОС Linux с помощью утилиты TLP.

1. Файл конфигурации будет записываться/модифицироваться в /etc/tlp.d/*.conf (далее "целевой конфиг").
1.1 Если при смене профиля уже существовал целевой конфиг, то он пересоздается с содержимым текущего конфига.
1.2 Для управления лимитом максимального заряда при питании от сети используется отдельный файл конфига и он настраивается отдельно, но аналогично основному конфигу.
2. Для смены режимов питания используется настраиваемая комбинация клавиш.
2.1 Имеется возможность циклической смены конфигов для настройки основного конфига.
2.2 Для настройки лимита максимального заряд необходимо выбрать нужную опцию в окне приложения.
